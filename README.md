# sportradar-assesment-ts

## Install dependencies

```bash
npm install
```

## Run tests

```bash
npm run test
```

## Run the application

```bash
npm run start
```

#### The Scoreboard is attached to `document.scoreboard`.
#### You can manually add matches to the scoreboard by calling `document.scoreboard.startMatch("Team 1", "Team 2")`.
